# bemora-agent-starter 🚀

Quick example of how to use bemora with AI agents!

## Install and run

```bash
npm install
npm install ../..  # if you want to use your local bemora build
node simple-agent.js
```

## What this does

This agent uses **bemora**'s 100+ tools to answer questions and take actions!
